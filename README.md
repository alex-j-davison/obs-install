# Overview

## Scope of script

Script does the following:-

* Update OS
* Install microk8s 
* Creates alias
* Download kubeinvaders ()
* Download Otel collector
* Update repos
* Setups kubeinvaders 
* Setups microk8s config 

## Installation steps

Steps:-

1. Clone repo

<code> git clone https://github.com/alex-j-davison/obs-install.git </code>

2. Change directory

<code> cd obs-install/ </code>

3. Change permissions on shell scripts to execution

<code> chmod +x step1.sh step2.sh step3.sh </code>

4. Run setup

<code> ./step1.sh </code>

<code> ./step2.sh </code>

<code> ./step3.sh </code>
